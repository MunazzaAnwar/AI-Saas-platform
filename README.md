# 🚀 AI SaaS Platform Landing Page

A dark, futuristic SaaS dashboard landing page for an AI-powered business platform — built with React, Tailwind CSS, and Framer Motion.

🌐 **Live Demo:** [munazzaanwar.github.io/AI-Saas-platform](https://munazzaanwar.github.io/AI-Saas-platform/)

---

## 📌 Overview

This landing page is designed for a premium AI SaaS product targeting modern startups and enterprises. It combines glassmorphism UI, gradient effects, and fluid Framer Motion animations to deliver the visual polish expected from platforms valued at millions of dollars.

---

## 🛠 Tech Stack

| Technology | Purpose |
|---|---|
| React | Component-based architecture |
| Tailwind CSS | Utility-first, dark-theme styling |
| Framer Motion | Page transitions, scroll animations, hover micro-interactions |
| React Icons | Consistent icon system |

---

## ✨ Features

### Design Language
- 🌑 Dark futuristic UI with glassmorphism panels
- 🌈 Animated gradient backgrounds and accent effects
- 💎 Premium startup aesthetics

### Sections
1. **Navbar** — Sticky, responsive navigation with CTA
2. **Hero** — Bold headline, animated gradient, floating dashboard preview
3. **Product Features** — Feature grid with animated icons
4. **Interactive Dashboard Preview** — Mock dashboard UI showcase
5. **Analytics Showcase** — Animated charts and metric cards
6. **Integrations** — Logo grid with hover effects
7. **Pricing Plans** — Tiered cards with highlighted recommended plan
8. **Testimonials** — Customer reviews with avatars
9. **FAQ** — Accordion-style questions
10. **Contact** — Simple CTA form
11. **Footer** — Links, social icons, copyright

### Animations (Framer Motion)
- Page entrance transitions
- Scroll-reveal on every section
- Floating card animations in hero
- Mouse-tracking hover effects
- Micro-interactions on buttons and cards
- Animated gradient overlays

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/munazzaanwar/AI-Saas-platform.git

# Install dependencies
cd AI-Saas-platform
npm install

# Start development server
npm start
```

Build for production:
```bash
npm run build
```
